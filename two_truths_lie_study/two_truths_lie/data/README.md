# Fact Database V1.0

This directory contains the fact database for the Two Truths and a Lie experiment.

## Structure

### `curated_facts.json` (18 facts)
Manually curated, highly verified facts across 6 traditional categories:
- science (3 facts)
- history (3 facts)
- biology (3 facts)
- geography (3 facts)
- technology (3 facts)
- culture (3 facts)

**Purpose:** Add recent facts here that are outside LLM training cutoffs for out-of-distribution testing.

### `raw/*.json` (81 facts)
LLM-generated facts across 11 expanded categories:
- historical_oddities (10 facts)
- scientific_discoveries (7 facts)
- cultural_traditions (7 facts)
- natural_phenomena (7 facts)
- animal_behaviors (3 facts)
- food_origins (10 facts)
- unlikely_inventions (10 facts)
- archaeological_mysteries (3 facts)
- forgotten_figures (7 facts)
- unexpected_connections (7 facts)
- sports (10 facts)

**Generated by:** 3 models (Claude Sonnet 3.7, GPT-4o, Gemini 2.5 Flash)

## Total Database

**99 facts** across **17 unique categories**

## Usage

The `FactDatabase` class automatically loads facts from both sources:

```python
from src.facts.database import get_default_facts

db = get_default_facts()
print(f"Total facts: {len(db)}")  # 99
print(f"Categories: {len(db.categories)}")  # 17
```

## Generation

To generate more facts:

```bash
# Generate 10 facts for a specific category
python -m src.facts generate --category sports --count 10 --output data/raw

# Generate 10 facts per category for all categories
python -m src.facts generate --count 10 --distribute-evenly --output data/raw

# View statistics
python -m src.facts stats --input data/raw
```
